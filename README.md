### 组件库使用及注意
```
该组件库为vue2组件库，对于vue3项目使用有兼容性问题

1.装依赖包 
npm install ndst-ui

2.在main.js中引入UI库 
import myUI from 'ndst-ui'
import 'ndst-ui/lib/ndst-ui.css

3.全局注册
Vue.use('myUI')
```
